> 本篇文章主要验证普通函数和内联函数的区别

Inline_function.cpp

```c++
__attribute__((always_inline))
inline void func() {
    int j = 0;
    for (int i = 0; i < 2; i++) {
        j++;
        if (j == 20) j *= 2;
    }
}

int main() {
    func();
    func();
    func();
    func();
    func();
    return 0;
}
```

`__attribute__((always_inline))`告诉编译器强制内联。

<!--more-->

outline_function.cpp

```c++
void func() __attribute__((noinline));
void func() {
    int j = 0;
    for (int i = 0; i < 2; i++) {
        j++;
        if (j == 20) j *= 2;
    }
}

int main() {
    func();
    func();
    func();
    func();
    func();
    return 0;
}
```

`__attribute__((noinline))`告诉编译器一定不要内联
